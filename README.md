# Wine Quality Prediction

## 📌 Overview

This project predicts wine quality using the Naive Bayes classification algorithm. It demonstrates data preprocessing, feature analysis, model training, and performance evaluation.

---

## 🎯 Objective

Build a classification model to predict the quality of wine based on its physicochemical properties.

---

## 📂 Dataset

- **Dataset:** Wine Quality Dataset
- **Problem Type:** Classification
- **Target Variable:** Quality

---

## 🤖 Machine Learning Model

- Naive Bayes Classifier

---

## ⚙️ Data Preprocessing

- Handling missing values
- Feature selection
- Feature scaling
- Train-Test Split

---
